# devops-capstone-project

![Build Status](https://github.com/josscui516/devops-capstone-project/actions/workflows/ci-build.yaml/badge.svg)

This project is part of a DevOps capstone where we build a customer account microservice for an e-commerce platform.

The service is implemented using Python Flask and provides a RESTful API to manage customer accounts, including create, read, update, delete, and list operations.

The project follows DevOps best practices such as version control, CI/CD, containerization, and cloud deployment.
